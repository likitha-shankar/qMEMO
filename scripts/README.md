# scripts

Automation scripts for setup, benchmarking, and reporting.

## Common Scripts

- `install_liboqs.sh` - build/install local liboqs dependency.
- `run_all_benchmarks.sh` - orchestrate benchmark execution.
- `analyze_results.py` - aggregate and compute summary metrics.
- `generate_report.py` - produce report-friendly outputs.
- `run_logged.sh` - wrapper for timestamped/logged command execution.
- `chameleon_setup.sh`, `setup_project.sh` - environment/bootstrap helpers.
