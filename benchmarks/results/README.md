# benchmarks/results

Benchmark output artifacts (CSV/log files and curated evidence bundles).

## Important Subsets

- `hybrid_matrix_apr18_final/` - canonical reviewer-safe April 18 matrix (final set).
- `hybrid_matrix_apr18/` - raw history including intermediate/retry attempts.
- `run_*` folders - timestamped historical benchmark snapshots.
- `cloud_repro_live/` - live cloud reproduction working artifacts.
- Other timestamped folders/files may contain exploratory runs.

## Interpretation

- Canonical aggregate numbers are summarized in `docs/RESULTS.md`.
- Run-level proof artifacts are captured in `docs/RUN_EVIDENCE.md`.
