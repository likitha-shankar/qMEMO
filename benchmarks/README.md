# Benchmarks

This folder contains standalone cryptographic performance benchmarks used in qMEMO.

## Contents

- `src/` - C source files for algorithm-level benchmarks.
- `bin/` - compiled benchmark executables (generated by `make`).
- `results/` - benchmark output artifacts (logs/CSVs, including canonical sets).
- `Makefile` - build and run entry points for benchmark binaries.

## Typical Usage

```bash
cd benchmarks
make clean
make all
make run
```

For consolidated interpretation, see `docs/RESULTS.md` and `docs/FINDINGS.md`.
